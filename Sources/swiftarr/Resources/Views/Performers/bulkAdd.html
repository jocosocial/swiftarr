Fetches a performer listing page, scrapes each linked performer page for their full profile data, and compares the results against performers already in the database. You will be shown a summary of changes before anything is applied. This may take a while as each performer page is fetched individually.

Sample mode fetches the list but then only one performer, letting you test the process before doing the full scrape.

Recommended workflow: Import from Sched first, fill in any known alternative names, then import from jococruise.com. Alternative names are used to match performers across sources. You will still probably need to do some manual massaging or verification.
